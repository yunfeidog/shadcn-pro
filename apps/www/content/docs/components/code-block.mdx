## Usage

```tsx showLineNumbers

```

```tsx showLineNumbers
<CodeBlock language="javascript">
  {`function greetUser(name) {
  console.log(\`Hello, \${name}!\`);
  return \`Welcome to our app, \${name}!\`;
}

// Usage
const message = greetUser("Alice");`}
</CodeBlock>
```

## Features

- **macOS-style Design**: Beautiful window with red, yellow, and green control dots
- **Syntax Highlighting**: Powered by Prism React Renderer with GitHub theme
- **Copy Functionality**: One-click copy with visual feedback
- **Language Display**: Shows the programming language in the window header
- **Responsive**: Works seamlessly across different screen sizes
- **Customizable**: Supports custom styling via className prop

## Props

### CodeBlock

| Prop        | Type     | Default        | Description                                  |
| ----------- | -------- | -------------- | -------------------------------------------- |
| `children`  | `string` | `-`            | The code content to display as a string      |
| `language`  | `string` | `"javascript"` | Programming language for syntax highlighting |
| `className` | `string` | `-`            | Additional CSS classes to apply to component |

## Supported Languages

The component supports syntax highlighting for 100+ programming languages via Prism.js:

- **Popular Languages**: JavaScript/TypeScript, Python, Java/Kotlin, C/C++/C#, Go/Rust, PHP/Ruby
- **Web Technologies**: HTML/CSS/SCSS, JSON/YAML/XML
- **Shell Scripts**: Bash/PowerShell, Shell
- **Databases**: SQL, GraphQL
- **And many more via Prism.js**

## Design Features

- **Window Controls**: macOS-style colored dots (red, yellow, green) in the header
- **Language Badge**: Displays the current programming language in the header bar
- **Copy Button**: One-click copy functionality with visual feedback
- **Theme**: Uses GitHub light theme from Prism React Renderer
- **Typography**: Monospace font with optimized line height for readability
